# One on One Quiz App using sockets
Quiz over the terminal using sockets !

This app uses python sockets library to connect two players and initiates a quiz between tham !

### How to use ?

**Open the termnial** ::

1. Start server 
```bash
python3 server.py
 ```
2. Start Client 1
```bash
python3 client.py localhost 8006
 ```
 3. Start Client 2
 ```bash
python3 client.py localhost 8006
 ```
